# Proposal: 通用多文件上传组件替换证件上传

## What
将合同签署等场景下的“证件上传”功能，统一为通用多文件上传组件：
- 不再区分证件类型、正反面等字段
- 仅允许上传图片类型文件，支持最多20张、单张最大10M
- 支持移动端上传、预览、删除
- 上传后仅保存图片URL数组
- 组件可复用于其他业务场景

## Why
1. 简化用户操作，减少无关字段，提升体验（参考 global/spec.md 移动端简化原则）
2. 满足实际业务只需上传图片的需求，避免冗余信息
3. 降低前后端复杂度，提升复用性和维护性
4. 便于后续扩展和统一管理上传逻辑
5. 对齐合同/全局规范，支持移动端、图片预览、删除等能力
