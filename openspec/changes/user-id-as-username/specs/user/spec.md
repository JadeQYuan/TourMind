## MODIFIED Requirements

### Requirement: 新增/编辑用户
系统 SHALL 支持通过表单新增用户和编辑用户信息（如姓名、手机号、角色），用户名（工号）不可编辑。新增用户时用户名由系统自动分配。

#### Scenario: 新建用户表单校验
- **WHEN** 管理员点击“新增用户”并填写表单后提交
- **THEN** 系统校验必填项，校验通过则保存新用户，用户名字段为只读且自动分配

#### Scenario: 新建用户时分配唯一工号
- **WHEN** 管理员新建用户
- **THEN** 系统分配唯一的六位数字工号，若已存在则自动递增到下一个可用值

#### Scenario: 编辑用户信息
- **WHEN** 管理员在用户列表点击“编辑”并修改信息后保存
- **THEN** 系统保存变更，用户名字段只读不可编辑

### Requirement: 用户列表分页查询
系统 SHALL 支持用户列表的分页展示，默认每页 20 条，并支持按工号、手机号、角色等字段模糊搜索。

#### Scenario: 按工号查询
- **WHEN** 管理员在搜索框输入工号并提交
- **THEN** 系统按工号过滤用户列表，显示匹配结果
