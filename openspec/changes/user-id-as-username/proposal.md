## Why

当前系统用户名为可编辑文本，存在重复、难以管理的问题。为提升用户管理规范性与系统安全性，需将用户名改为唯一的六位数字工号，自动递增分配，且不可重复、不可编辑。

## What Changes

- 新用户创建时，系统自动分配六位数字工号作为用户名，递增且唯一
- 用户名（工号）在用户列表中展示，并支持查询
- 用户名（工号）可用于登录
- 用户名（工号）创建后不可修改

## Capabilities

### New Capabilities
- `user-id-as-username`: 用户名采用六位数字工号，自动递增、唯一、不可编辑，可用于登录、展示和查询

### Modified Capabilities
- `user`: 用户名字段由可编辑文本改为系统分配的六位数字工号，且不可修改，支持列表展示、查询和登录

## Impact

- 涉及后端 user 模型、schemas、API 逻辑
- 涉及前端用户管理页面、登录逻辑、API 调用
- 涉及数据库 user 表结构及相关迁移
