## Purpose
供应商管理：支持供应商的分页查询、搜索、新增、编辑、禁用、查看详情等基础操作。

## Fields
| 字段名       | 中文名     | 类型     | 约束/说明                         |
|------------|----------|---------|----------------------------------|
| id         | 供应商ID  | int     | 主键，自增                        |
| name       | 供应商名称 | string  | 必填，唯一，1-64 字符              |
| type       | 类型      | string  | 必填，枚举：地接社/酒店/交通/其他   |
| contact    | 联系人    | string  | 必填，1-32 字符                    |
| phone      | 联系电话  | string  | 必填，手机号或座机格式校验         |
| email      | 邮箱      | string  | 可选，邮箱格式校验                 |
| address    | 地址      | string  | 可选，最长 128 字符                |
| status     | 状态      | string  | 必填，枚举：enabled/disabled       |
| remark     | 备注      | string  | 可选，最长 128 字符                |
| created_at | 创建时间  | string  | 创建时间，ISO8601                  |
| updated_at | 更新时间  | string  | 更新时间，ISO8601                  |

## Requirements


### Requirement: 供应商列表分页查询
系统 SHALL 支持供应商列表的分页展示，默认每页 20 条，并支持按供应商名称、类型、联系人等字段模糊搜索。

#### Scenario: 分页加载供应商列表
- **WHEN** 用户进入供应商管理页面
- **THEN** 系统自动加载第一页供应商数据，显示分页控件，可切换页码加载对应数据

#### Scenario: 输入关键词搜索
- **WHEN** 用户在搜索框输入关键词并提交
- **THEN** 系统按关键词过滤供应商列表，显示匹配结果


### Requirement: 新增/编辑供应商
系统 SHALL 支持通过表单新增供应商和编辑供应商信息（如名称、类型、联系人、电话等），名称不可与其他供应商重复。新增供应商时必填项校验，名称唯一。

#### Scenario: 新建供应商表单校验
- **WHEN** 用户点击“新增供应商”并填写表单后提交
- **THEN** 系统校验必填项，校验通过则保存新供应商，失败则显示错误提示

#### Scenario: 编辑供应商信息
- **WHEN** 用户在供应商列表点击“编辑”并修改信息后保存
- **THEN** 系统保存变更，名称字段唯一性校验

### Requirement: 供应商状态变更
系统 SHALL 支持管理员在供应商列表中对供应商进行禁用和启用操作，被禁用供应商不可被新订单选择，启用后可正常选择。

#### Scenario: 禁用供应商
- **WHEN** 用户在供应商列表点击“禁用”按钮
- **THEN** 系统将该供应商状态设为禁用，后续新订单无法选择该供应商

#### Scenario: 启用供应商
- **WHEN** 用户在供应商列表点击“启用”按钮
- **THEN** 系统将该供应商状态设为启用，可正常选择

### Requirement: 查看供应商详情
系统 SHALL 支持查看供应商详细信息，展示所有字段。

#### Scenario: 查看详情抽屉
- **WHEN** 用户点击供应商列表的“查看”按钮
- **THEN** 系统右侧弹出抽屉，展示该供应商的全部信息
